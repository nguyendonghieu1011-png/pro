chuyển đổi nhánh : git checkout `tên nhánh`

code chỉ thay đổi ở nhánh đang tương tác , còn khi mà chuyển đổi sang nhánh khác => code không thay đổi

đẩy nhánh lên git: git push -u origin `tên nhánh`

kiểm tra nhánh : git branch

xóa nhánh trên vscode : git branch -d 'ten nhanh'
xóa nhánh trên git: git push -d origin 'ten nhanh'
